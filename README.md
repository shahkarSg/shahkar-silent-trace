**Shahkar Silent Trace** 
is a lightweight Windows security tool designed to silently detect and trace malware footprints and unwanted files.  
This project was initially known as **Shahkar Anti Malware** during its early development.  
The official and final release is now branded as **Shahkar Silent Trace**.

- Detect and clean traces of malware or unwanted files in:
  - Processes
  - Network connections
  - Registry keys
  - Hijacks
  - Tasks
  - Services
  - Drivers
  - Disk-level traces

- File analysis with **VirusTotal**

- Display and repair damages caused by malware, including:
  - System Restore
  - Task Manager
  - Registry Editor
  - Folder Options
  - Other critical Windows components
